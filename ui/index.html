Quick Start Guide

  1. Run supabase_schema.sql in Supabase Dashboard → SQL Editor
  2. Go to Settings → fill LiveKit, Gemini, Vobiz credentials → Save each
  3. Click Create SIP Trunk in LiveKit in Vobiz section
  4. Go to AI Prompt → customize for your business → Save
  5. Go to Agents → create a profile with your business prompt
  6. Go to Single Call → test with your own number
  7. Hear the AI speak within 5-10 seconds!
